내가 궁극적으로 만들고자 하는 것.

사용자가 접속하면 `LoadingPage.png` 느낌의 트렌디한 로딩 화면이 뜬다.
로딩이 끝나면 `landingpage.png` 스타일의 홈 화면이 나온다.

핵심 제품 목표는 다음과 같다.
1. 사용자가 **도면 업로드 없이** 방을 빠르게 생성한다. (IKEA Kreativ 스타일)
2. 생성한 방 안에서 3D 데스크테리어를 배치/수정/저장한다.
3. 작업 결과를 커뮤니티에 게시하고, 다른 사용자가 동일 장면을 3D 읽기 전용 뷰어로 확인한다.

필수 UX:
- 빌더: 방 형태/치수/문창/스타일 선택
- 에디터: 가구 드래그/회전/정렬, 재질/조명 조정
- 공유: publish 후 토큰 링크 생성
- 커뮤니티: gallery/community에서 3D 장면 조회 + 제품 정보 확인

품질 목표:
- PBR + HDR + Post FX 기반의 고급스러운 시각 품질
- 데스크/소품 중심 카탈로그 완성도
- Blender 원본 기반 자산 파이프라인 유지
- 저장/복원/공유 시 동일한 sceneDocument 계약 유지

중요:
- 과거 floorplan 인식/2D 보정/자동 3D 변환 파이프라인은 제품 메인 경로에서 제거한다.
