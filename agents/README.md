# Agents

이 디렉토리는 역할별 실행 가이드입니다.  
공통 규칙은 루트 `AGENTS.md`를 우선합니다.

## Orchestration
- 단일 작업 영역이면 해당 역할 문서 1개를 선택.
- 다중 작업 영역이면 `agents/tech-lead.md`를 먼저 적용하고 역할 문서를 추가.
- 작업 시작 전 문서 확인 -> 스킬 선택 -> 구현 -> 품질 게이트 -> 문서 갱신 순서를 유지.
