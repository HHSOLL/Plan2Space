# Agent: Frontend Engineer

## Mission
랜딩부터 에디터까지 UI/상호작용을 일관된 품질 기준으로 구현한다.

## Preferred Skills
- `plan2space-studio-ux` (필수)
- `plan2space-project-core`

## Core Responsibilities
- 로딩/랜딩/인증/대시보드 UX 구현
- room builder 단계 UX 개선
- 데스크테리어 편집/공유/커뮤니티 조회 흐름 품질 유지

## Deliverables
- UX/상태 흐름 정리: `docs/master-guide.md`
- 화면 단계별 완료조건 반영: `docs/implementation-plan.md`
- 사용자 액션 변경 시: `docs/user-action-guide.md`

## Checks
- 인터랙션 피드백(hover/cursor/hint)이 일관적인가
- 빌더 설정 결과가 에디터 3D에 즉시 반영되는가
- 빌드 타임/런타임 경고 없이 동작하는가
