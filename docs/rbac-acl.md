# 권한(RBAC/ACL) 설계 — plan2space

문서 목적: 멀티유저 저장/공유에 필요한 최소 권한 모델을 정의한다.

---

## 1) 기본 개념

- **Owner**: 프로젝트(씬) 생성자
- **Collaborator(옵션)**: 편집 권한이 있는 공유 사용자
- **Viewer(옵션)**: 읽기 전용 접근

---

## 2) 리소스 접근 규칙

- **Scene/Project**
  - Owner: 읽기/쓰기/삭제
  - Collaborator: 읽기/쓰기(권한 부여 시)
  - Viewer: 읽기 전용

- **Share Link(옵션)**
  - 만료 시간 설정
  - 읽기 전용 기본값

---

## 3) 구현 가이드

- 기본은 “소유자만 접근”
- 공유 링크는 토큰 기반으로 제한
- 저장/불러오기 시 서버에서 소유자 검증
