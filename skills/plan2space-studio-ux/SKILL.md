# Plan2space Studio UX

## Workflow
1) Review `new_guideline/landingpage.png`, `new_guideline/LoadingPage.png`, and `new_guideline/pov.jpg` for target feel.
2) Read `docs/master-guide.md` and `docs/3d-visual-engine.md` before changing UI or interaction patterns.
3) Preserve the two-mode camera system and interaction cues.

## UX Rules
- Landing uses keypad-style navigation and clear brand focus.
- Auth uses a 3D monitor-style popup (not a flat modal).
- Interactive objects change cursor state on hover.
- Top view supports drag/drop and material swaps; walk mode supports WASD + click actions.
